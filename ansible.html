Ansible


Ansible é uma linguagem de configuração para automatização e gerenciamento de configuração de sistemas. Como instalação de programas e outras tarefas envolvendo CD/CI, entrega continua/ integração continua.
Nesse documento abordaremos a ferramenta apenas no modo de salvar arquivos de configurações de sistemas. Criar modelos de instalação de novas máquinas, configurações de instalação do servidores. Criar arquivos de configuração de kubernetes e outros serviços a forma de execução do Ansible.

Instalação


Sendo uma aplicação open-source, o Ansible pode ser encontrado no repositorio oficial de várias distribuição linux. Podendo ser optido atraves da instalação padrão de pacotes.
'sudo apt install ansible'(ubuntu,debian based machines)
Outros modos de instalar podem ser optidos via Documentação Oficial .
Uma vez instalado o proximo passo é editar o arquivo de configuração do ansible para as suas necessidades.
