Docker


Utiliza o conceito de containers, para fornecer microserviços. Ao inves de virtualizar uma imagem de um SO inteira, o docker utiliza o kernel da máquina em que está instalado e cria a virtualização apenas do serviço. Tornando mais rápido e fácil o descontrução e a reconstrução de serviços.
Utilizando o Docker Hub temos diversas images disponiveis para a utilização dos serviços. Mas podemos sempre criar novos containers a partir das imagens com a personilização necessária.
As configurações do docker são feitas de maneira bem simples. Podemos confirar um Dockerfile que seguindo o padrão consegue realizar todas as instalações necessárias.
